**Read profile**
----
Returns json data about a logged user.

* **URL**

  /users/me

* **Method:**

  `GET`

*  **URL Params**

   **Required:**

    - Authorization Bearer Token
    - User's ID

* **Data Params**

  None

* **Success Response:**

    * **Code:** 200 <br />

* **Fail Response:**

    * **Code:** 401 <br />
